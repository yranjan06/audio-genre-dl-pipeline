# DL & GenAI Project (Messy Mashup)

Project scaffold for the Messy Mashup genre classification project.

- Author: ranjanyadav
- Structure: data/, notebooks/, scripts/, models/, reports/

Quick start
1. Create and activate a virtual environment:

```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

2. Do not commit `data/raw/` to GitHub. Place your downloaded Kaggle data under `data/raw/`.

3. Run notebooks in `notebooks/` for EDA and baseline experiments.

W&B project: create `YourRollNo-t12026` and use in `scripts/train.py`.
